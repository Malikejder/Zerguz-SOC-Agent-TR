# Zerguz-SOC-Agent
Multi-threading mimariyle çalışan, ağ tabanlı saldırı tespiti (NIDS) ve uç nokta izleme (EDR) yeteneklerini birleştiren modüler SOC ajanı. BPF filtreleme, port tarama, ICMP flood, DNS tünelleme anomalileri tespiti ve otomatik PCAP kanıt toplama (IR) özellikleriyle SIEM platformları için uyumlu loglar üretir.
